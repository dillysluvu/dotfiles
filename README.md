## Installation

1. Initialize a directory for configuration data:
    ```sh
    mkdir -p ~/Projects
    cd ~/Projects
    ```

2. Clone my config:
    ```sh
    git clone https://github.com/dillysluvu/dotfiles.git
    ```

3. Run the setup program:
    ```sh
    chmod +x setup.sh
    ./setup.sh
    ```
