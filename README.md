## Installation

1. Clone the configuration repository:
    ```sh
    git clone https://github.com/dillysluvu/dotfiles.git
    ```

2. Run the setup script:

    - For Fedora Linux or Fedora-based distributions:
      ```sh
      chmod +x setup.sh
      ./setup.sh
      ```

    - For Arch Linux or Arch-based distributions:
      ```sh
      chmod +x setup1.sh
      ./setup1.sh
      ```
